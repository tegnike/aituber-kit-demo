# クイックスタート

## インストール

```bash
git clone https://github.com/tegnike/aituber-kit.git
cd aituber-kit
npm install
```

## 環境変数の設定

1. `.env.example`を`.env.local`にコピー

```bash
cp .env.example .env.local
```

2. 必要な環境変数を設定

## 開発サーバーの起動

```bash
npm run dev
```
